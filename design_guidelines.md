# Hindu Umiya Maa Temple Website Design Guidelines

## Design Approach: Cultural Heritage-Inspired

**Selected Approach**: Reference-based design drawing inspiration from traditional Hindu temple architecture and modern spiritual websites, emphasizing reverence, accessibility, and cultural authenticity.

**Key Design Principles**:
- Sacred simplicity with intentional visual hierarchy
- Cultural authenticity without overwhelming complexity  
- Peaceful, meditative user experience
- Accessibility for all age groups and technical abilities

## Core Design Elements

### A. Color Palette

**Primary Colors** (Dark Mode):
- Sacred Saffron: 25 85% 65% 
- Deep Temple Red: 0 70% 45%
- Warm Gold Accent: 45 90% 70%

**Primary Colors** (Light Mode):
- Sacred Saffron: 25 75% 55%
- Deep Temple Red: 0 60% 40% 
- Warm Gold Accent: 45 80% 60%

**Supporting Colors**:
- Pure White/Cream: 45 15% 95%
- Soft Ivory: 45 20% 88%
- Deep Charcoal: 220 15% 15%

### B. Typography

**Primary Font**: Crimson Text (Google Fonts) - elegant serif for headings and sacred text
**Secondary Font**: Inter (Google Fonts) - clean sans-serif for body text and navigation
**Accent Font**: Noto Sans Devanagari (Google Fonts) - for any Sanskrit/Hindi text

**Hierarchy**:
- Hero Titles: Crimson Text, 3xl-4xl, font-bold
- Section Headers: Crimson Text, 2xl-3xl, font-semibold  
- Body Text: Inter, base-lg, font-normal
- Captions: Inter, sm, font-medium

### C. Layout System

**Spacing Primitives**: Tailwind units of 4, 8, 12, and 16 (p-4, m-8, gap-12, py-16)
- Consistent 4-unit grid system
- Generous whitespace for peaceful feeling
- Section padding: py-16 for major sections, py-8 for subsections

### D. Component Library

**Navigation**: 
- Clean horizontal nav with subtle hover states
- Sacred symbol integration in logo area
- Sticky header with transparency effects

**Hero Section**:
- Large background image of temple with gradient overlay
- Centered content with outlined buttons (blurred backgrounds)
- Prayer times or live status indicator

**Content Cards**:
- Subtle shadows and rounded corners (rounded-lg)
- Image overlays with gradient treatments
- Hover effects with gentle transitions

**Forms**:
- Rounded inputs with warm border colors
- Sacred iconography for form sections
- Accessible error states and validation

**Gallery Components**:
- Masonry-style image grids
- Lightbox functionality for detailed viewing
- Upload interfaces for admin content management

### E. Animations

Minimal, purposeful animations:
- Gentle fade-ins for content sections
- Smooth hover transitions (200ms duration)
- Sacred symbol subtle rotations or pulses
- Page transitions with peaceful timing

## Images

**Hero Image**: Large, high-quality photograph of the Umiya Maa Temple exterior or interior sanctum as the primary hero background. Should convey grandeur and spiritual atmosphere.

**Supporting Images**:
- Temple architecture details for About/History sections
- Ceremony and festival photographs for Events page
- Devotee community images for Gallery
- Sacred symbols and traditional motifs as decorative elements
- Deity images and temple artifacts for cultural context

**Image Treatment**: Warm color grading, respectful composition, high contrast for text overlays, consistent aspect ratios across sections.

**Gallery Organization**: Categorized by festivals, daily ceremonies, temple architecture, and community events with intuitive filtering and search capabilities.
